* Basic Network Sniffer

* Description
This project is a basic network sniffer built using Python for Cyber Security internship Task 1.
It captures network packets and displays useful information such as source IP, destination IP, and protocol type.

* Tools & Technologies
- Python
- Scapy library

*Features
- Captures live network packets
- Identifies TCP and UDP protocols
- Displays source and destination IP addresses
